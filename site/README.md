# Marketing Site (GitHub Pages Ready)

Static marketing site source for the protocol.

## Files

- `site/index.html`
- `site/styles.css`
- `site/script.js`

## Preview Locally

Use any static server from repository root:

```bash
npx serve site
```

or

```bash
python -m http.server --directory site 4173
```

## Publish to GitHub Pages (later)

1. Push the repository to GitHub.
2. In repository settings, enable Pages.
3. Choose deployment source:
- `main` branch
- `/site` folder
4. Save and wait for build completion.
