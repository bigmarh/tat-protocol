Open-source protocol SDK

Issue, transfer, and verify tokenized access over Nostr.

TAT Protocol gives teams a complete TypeScript stack for forge, pocket, NWPC transport, gate verification, and booth checkout workflows.

Read the SDK docs Adoption guide

Forge + Token

Mint fungible and non-fungible assets with spent-token safeguards and issuer controls.

Pocket + Storage

Track balances, hold assets, and sign transfer flows with state persistence across sessions.

NWPC Transport

Encrypted request/response RPC over Nostr relay infrastructure with route metadata support.

Gate + Booth

Implement access checks, challenge proofs, catalogs, invoices, and on-protocol payments.
